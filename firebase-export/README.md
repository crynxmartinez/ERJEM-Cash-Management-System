# Firebase Export Tool

Standalone local tool to export all Firebase data to JSON files.

## Usage

1. Install dependencies:
```bash
npm install
```

2. Run export:
```bash
npm run export
```

3. Find your data in the `./data/` folder:
   - `users.json`
   - `branches.json`
   - `transactions.json`
   - `userBranches.json`
   - `all-data.json` (combined)
